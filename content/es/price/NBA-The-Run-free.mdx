## Cómo evitar descargas falsas y estafas de códigos

Cuando una palabra clave como **NBA The Run gratis** se vuelve tendencia, los estafadores se mueven rápido. La mayoría de actores maliciosos se apoyan en la urgencia (“¡últimas 20 claves!”), archivos de lanzador falsos y phishing de cuentas.

### Lista de señales de alerta

| Señal de alerta | Cómo se ve | Respuesta segura |
|---|---|---|
| **Instalador fuera de plataforma** | “EXE directo con bypass” | Saltar y eliminar |
| **Estafa de verificación de cuenta** | “Inicia sesión con Steam aquí” | Usa solo páginas oficiales de Steam |
| **Lotes de claves demasiado buenos para ser verdad** | “Claves ilimitadas por $5” | Reportar, no interactuar |
| **Distribución solo por DM** | “Envía ahora el código del autenticador” | Bloquear y reportar |
| **Sin reglas públicas** | “Ganadores elegidos en secreto” | Tratar como no confiable |

### Flujo seguro para reclamar

1. Confirmar identidad e historial de publicaciones del anfitrión del sorteo  
2. Verificar requisito de plataforma (normalmente Steam para claves tempranas de PC)  
3. Canjear directamente en el cliente/interfaz oficial  
4. Activar autenticación de dos factores antes de reclamar cualquier cosa  
5. Ignorar “ayudantes de activación” de terceros

Si sigues este marco, puedes buscar oportunidades de **NBA The Run gratis** minimizando riesgos.

## NBA The Run Rollback Netcode Setup: Best Connection Practices

Even excellent online infrastructure benefits from good local setup. If you want clean reads in competitive games, treat your network like part of your build.

### 1) Prioritize Wired Ethernet

Wi-Fi introduces interference, especially in apartment-heavy areas. Ethernet reduces jitter and packet loss, which can matter more than raw download speed.

### 2) Stabilize Local Network Traffic

Pause large uploads, cloud backups, and background streaming on shared networks during ranked sessions. A family member’s 4K stream can spike your latency at the worst moment.

### 3) Choose Region-Smart Matchmaking

If the game provides region filters or connection-quality indicators, use them for serious play. Broader regions are fine for casual sessions, but ranked consistency improves with tighter routing.

### 4) Optimize Your Platform Settings

Enable low-latency display modes, game mode on TV/monitor, and wired controllers when possible. Netcode improvements can be undermined by local display lag.

| Setup Element | Recommended Baseline (2026) | Competitive Target |
|---|---|---|
| **Connection type** | Wired Ethernet | Wired + QoS router profile |
| **Ping to game servers** | Under 60 ms | Under 35 ms |
| **Jitter** | Under 10 ms | Under 5 ms |
| **Packet loss** | 0–1% | 0% |
| **Display latency** | TV Game Mode | Low-latency gaming monitor |

> **Warning:** Don’t chase only “download speed.” Netcode quality depends more on latency stability, jitter control, and packet consistency.

## Troubleshooting NBA The Run crossplay issues quickly

When cross-platform play fails, run diagnostics in layers: account, permissions, network, then server health. Avoid random setting changes; use a clean sequence.

| Problem | Most Likely Cause | Fast Fix | Escalation Path |
|---|---|---|---|
| **Can’t invite friend on another platform** | Account not linked correctly | Re-link account and restart client | Submit support ticket with platform IDs |
| **Invite sends but lobby join fails** | Privacy/cross-network permissions | Enable cross-network play in console settings | Check family safety controls |
| **Very long queue times** | Region mismatch or disabled crossplay | Set region to Auto, confirm crossplay on | Queue at peak local hours |
| **Desync or lag spikes** | High jitter/Wi-Fi instability | Switch to wired, reboot router | Use QoS or contact ISP |
| **Ranked queue unavailable with party** | Mode restrictions by input/rank gap | Align inputs and rank eligibility | Split party for placement matches |
| **Progress not appearing on second platform** | Cross-progression sync delay | Relog and wait sync window | Verify account linkage history |

### Quick 90-second health check before each session

1. Confirm everyone is on the **same patch version**.
2. Verify **crossplay toggle** stayed enabled after last restart.
3. Ask each party member to check platform privacy permissions.
4. Run one private lobby join test.
5. Start in casual mode, then move to ranked.

This workflow dramatically reduces failed queues in **NBA The Run crossplay** sessions.

> **Tip:** If the party leader changes, re-check queue settings. Some games inherit filters from the new host, which can quietly change eligibility.

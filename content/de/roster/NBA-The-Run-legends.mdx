## So baust du ein siegfähiges Legends-Squad im The Run World Tournament

Viele Spieler werden in **NBA The Run legends** stürzen, indem sie große Scorer stapeln. Das ist spannend, erzeugt aber meist Defensivlücken und inkonstante Possessions. Nutze stattdessen Rollenlogik.

### Empfohlene Lineup-Grundstruktur

| Slot | Primäre Funktion | Idealer Trait-Mix | Frühe Priorität |
|---|---|---|---|
| Lead Creator | Offense initiieren | Ballhandling + Übersicht + Passing unter Druck | **Sehr hoch** |
| Wing-Scorer | Halbfeld-Sets abschließen | Shot-Creation + Shooting in Bewegung | **Hoch** |
| Defensiv-Stopper | Top-Bedrohungen stören | Laterale Geschwindigkeit + Contest-Timing | **Hoch** |
| Glue Forward | Rebounds + Screens + Switches | Stärke + Spielverständnis + Ausdauer | **Mittel-hoch** |
| Utility-Bankspieler | Tempo je nach Matchup ändern | Spezialisierte Fähigkeit (Shooting/Defense) | **Mittel** |

### Chemie-Regeln, die du befolgen solltest

1. **Possession-Typen ausbalancieren**  
   Habe mindestens zwei Spieler, die im Fastbreak kreieren können, und einen verlässlichen Halbfeld-Finisher.

2. **Schwächen nicht doppeln**  
   Wenn zwei Kernspieler schwache On-Ball-Verteidiger sind, füge sofort einen Pressure-Defender hinzu.

3. **Nutzungshierarchie festlegen**  
   Dein bester Scorer sollte oft den Ball bekommen, aber deine zweite Option muss sich wohlfühlen, die Offense unter Druck zu initiieren.

4. **Baue zuerst deinen „Floor“**  
   Defense, Rebounding und Turnover-Kontrolle machen deine schwachen Spiele überlebbar.

Für die meisten Spieler ist der beste Ansatz in **NBA The Run legends** in der ersten Woche Konstanz statt Spektakel. Flair kannst du hinzufügen, sobald dein Basissystem stabil ist.

### Praktische Checkliste

- Definiere deinen **primären Scorer**, **sekundären Creator** und **Defensiv-Kapitän**
- Wähle zwei „sichere“ Plays für späte Clock-Possessions
- Erstelle eine schnelle Timeout-Anpassungsregel:
